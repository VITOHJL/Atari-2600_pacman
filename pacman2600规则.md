# Pacman 游戏规则

## 生命

总共4次，见左下角显示。每次吃豆人吃掉迷宫中所有的豆子（video wafers），就会获得额外一条生命和一个充满新豆子的新迷宫。
在迷宫的角落附近，有四个闪烁点，被称为**能量丸**，为吃豆人提供了暂时吃掉敌人的能力。敌人会变成深蓝色，逆转方向，移动速度也会变慢。当敌人被吃掉时，它的眼睛会保留并回到中间的盒子中，在那里它会重新变成正常的颜色。鬼魂会变成粉红色/闪烁白色（查到了两种不同的说法），表示他们即将再次变得危险，最后变回原来的颜色。敌人保持脆弱状态的时间也会随着游戏的进展而变短。在后期阶段，敌人会直接闪动，绕过蓝色，这意味着他们只能在很短的时间内被吃掉，尽管当能量颗粒被吃掉时，他们仍然会改变方向；即使在后期，鬼也不会变成可食用的（即它们不会改变颜色，但仍然会让吃豆人失去生命），但它们仍然会逆转方向。
## 维生素

维生素是巢穴中央两个相交的矩形。它们只出现几分钟，然后消失又出现。不同游戏难度中，能量丸持续时长和维生素出现时长不同。
地图上方的开口和下方开口是相通的，进入开口处需要消耗一两秒才能从另一个出口出来，可以利用这两个开口来躲避幽灵（幽灵也可以使用相同的路线）。吃能量丸之前，尽可能多吃一些豆子。吃能量丸之前，站在靠近它的角落里，诱使幽灵靠近。
鬼整体朝pacman靠近，但是在移动中有一定的随机性。
## 技术说明

Atari 2600 没有像样的像素碰撞检测，因此 Pac-Man 和鬼撞到一起时，程序不是检查像素重叠，而是使用位置区间。如果 Pac-Man 的 X/Y 落在鬼的"危险区域范围"，即判定死亡。这导致：有时靠得很近却没死，有时远一点也被判死。这不是 bug，而是硬件限制导致的大颗粒碰撞框。
